# ⏳ TimeCapsule AI

A futuristic **Angular application** that allows users to create digital time capsules containing messages, emotions, and reflections that unlock in the future.

## ✨ Features

• Create digital time capsules
• Set future unlock dates
• Mood-based capsule visualization
• Real-time countdown timer
• AI-inspired chat interaction
• Memory insights dashboard

---

## 🧠 Project Concept

TimeCapsule AI works like a **digital memory vault**.

Users can write messages today and unlock them in the future.

Each capsule contains:

• Title
• Message
• Emotion / Mood
• Unlock date
